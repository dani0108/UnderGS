# UnderGS: Understanding Evolving Graph Structures for Large Discrete-Time Dynamic Graph Representation

This repository contains the code, configuration, and data scripts used in the implementation and evaluation of the proposed **UnderGS** framework.

UnderGS is a scalable and backbone-agnostic framework designed to understand and leverage evolving graph structures in **Discrete-Time Dynamic Graphs (DTDGs)**. It integrates a temporal influence scoring mechanism and an online temporal-cohesive neighbor store to support temporal learning without additional modules.

## 🔧 Contents

- `src/`: Source code for the model and utilities
- `configs/`: Example configuration files for training and evaluation
- `scripts/`: Training and evaluation scripts
- `data/`: Scripts to download or preprocess datasets
- `README.md`: Project overview (this file)
